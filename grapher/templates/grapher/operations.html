Some basic JavaScript math operations

JavaScript Description
Math.E Euler's constant, approximately 2.718.
Math.PI Ratio of circumference of a circle to its diameter, approximately 3.14.
Math.abs(x) Returns the absolute value of x.
Math.ceil(x) Returns the smallest integer greater than or equal to x.
Math.cos(x) Returns the cosine of x.
Math.exp(x) Returns E^x, where E is Euler's constant.
Math.floor(x) Returns the largest integer less than or equal to x.
Math.log(x) Returns the natural logarithm of x.
Math.log10(x) Returns the base 10 logarithm of x.
Math.max(x1,x2,...) Returns the largest of its arguments.
Math.min(x1,x2,...) Returns the smallest of its arguments.
Math.pow(x,y) Returns x to the power of y.
Math.random() Returns a pseudo-random number between 0 and 1.
Math.sin(x) Returns the sine of x.
Math.sqrt(x) Returns the positive square root of x.
Math.tan(x) Returns the tangent of x.
